#  WinRAR 特性：

 * WinRAR 引入了一个原创的压缩算法。它对可执行文件，对象库，大文本
#   文件等提供了很高的压缩率。

 * RAR 格式在内部不限制压缩文件和文件的大小，也不限制压缩文件内文件
#   的数量。这些值可能受文件系统、操作系统以及可用内存的限制。

 * WinRAR 对 RAR 和 ZIP 2.0 格式压缩文件提供完整支持，并可以解压 7Z、
#   ACE、ARJ、BZ2、CAB、GZ、ISO、JAR、LZH、TAR、UUE、XZ、Z 压缩文件。

 * WinRAR 提供了经典交互式 Windows 界面和命令行界面。

 * WinRAR 提供了创建“固实”压缩文件的功能，这种格式相对于普通方式能
#   够使压缩率提高10% - 50%，当压缩大量小文件时更加显著。

 * WinRAR 提供了创建可定制的自解压压缩文件的功能。

 * WinRAR 提供了创建多卷自解压格式压缩文件的功能。

 * 可选的恢复记录，能够保护压缩文件防止损坏。

 * WinRAR 提供了大量服务功能，例如加密文件、添加压缩文件注释和保留 
   NTFS 文件的安全性。
